# spacetraders

A CLI command for operating your ships for spacetraders.io.

## Setup

```bash
# prepare your agent info at .env
cp .env-sample .env

# register agent, and copy the access token to .env
./scripts/register_agent.sh
```

## Prerequisites

- Go >= 1.20
